# lets-goweb

一起开始 `golang` 的web开发吧，通过标准库的源码进行讲解如何构建一个web程序，以及第三方框架的使用示例讲解。
笔者是 `java` 开发学习 `golang` 的，文中如有错误还请指出，不过使用 `go` 之后发现很多东西比java要方便多了 😂
